Olá, seja bem-vindo ao nosso sistema de controle de ponto.

Nosso sistema registra os pontos dos funcionários que seram cadastrados. Todos os registros devem ser efetuados tendo como base os funcionários já cadastrados, ou seja, caso você ainda não esteja cadastrado não será possível registrar os seus pontos. O cadastro, tanto dos pontos como do funcionário, deve ser feito pelo supervisor e/ou responsável pelo setor.

Assim que o funcionário for cadastrado, ele recebera um email informando e um resumo dos dados cadastrados. O mesmo vale para os pontos, e para qualquer alteração ou exclusão de dados. Todas as ações nos cadastros serão notificadas para o funcionário que esta sendo afetado.

O sistema possui também uma página de relatórios, caso o responsável pela área necessite realizar um levantamento dos funcionários ou pontos, esses dados são enviados no email informado no formato .CSV.
